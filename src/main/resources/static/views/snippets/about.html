Green-Box

Details

  • Class that must be run for open the server -> GreenBoxApplication.java;
  • URL that must be accessed for open the homepage on local server -> http://localhost:8080;
  • URL that must be accessed for open the homepage on heroku server -> https://green-box-si.herokuapp.com;
  • Use the branch heroku-release for run application;
  • Some functionality require double refresh for appear on front-end;

Data Restrictions

  • User Name must be between 4 and 15 characters, be letters, numbers and "-" or "_";
  • Email must be valid;
  • Password must have 6 or more characters at least, 1 letter and number;
  • Folders and files names, must be at least 1 character between letters, numbers and "_" (or spaces);
